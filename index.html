⚙️ Torn API — Terms of Service (Summary)

The full Torn API ToS is available at torn.com/api.html. Key points:

Key security: You must keep your API key confidential. Never share it with untrusted tools. This tool only sends your key to api.torn.com.

Rate limits: Up to 100 requests per minute per key. This tool makes at most 3 requests per load and stays well within limits.

Acceptable use: Read-only. No passwords are ever requested. This tool does not provide competitive advantage — it only reads data you already have access to as a director.

Logging: Torn logs all API requests. You can review your key's usage in Torn → Settings → API.

Access levels: Limited Access is sufficient. Full Access is not required or requested.
